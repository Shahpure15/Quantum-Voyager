Welcome to Quantum Voyager

An immersive platform for exploring quantum physics through interactive 3D simulations.

Interactive Experiments

  • Carbon Atom Model

    Experience a dynamic 3D visualization of a Carbon-12 atom:

    • Observe electron orbitals and quantum shells
    • Visualize wave-particle duality
    • Explore from femtometer to nanometer scale
    • Interact with electron probability clouds
  • Quantum Entanglement

    Explore the fascinating world of quantum entanglement:

    • Visualize entangled particle pairs
    • Observe quantum state correlations
    • Experiment with measurement effects
    • Understand Bell's inequality
  • Quantum Computing

    Dive into quantum computation fundamentals:

    • Interact with quantum gates and circuits
    • Visualize qubit states on Bloch sphere
    • Experiment with quantum algorithms
    • Explore quantum superposition
  • Double-Slit Experiment

    Witness the core principles of quantum mechanics:

    • Observe wave-particle duality in action
    • Control particle emission rates
    • View interference patterns develop
    • Explore measurement effects

Platform Features

  • Interactive Controls: Rotate, zoom, and manipulate 3D models with intuitive mouse controls
  • Real-time Scaling: Dynamic scale indicator showing quantum realm measurements
  • Immersive Audio: Atmospheric background music enhancing the experience
  • Educational Tooltips: Detailed explanations of quantum phenomena
  • Responsive Design: Seamless experience across different devices

Getting Started

  • Use mouse to rotate and zoom the 3D models
  • Navigate between experiments using the buttons above
  • Watch the scale indicator to understand size references
  • Toggle audio for an immersive experience

Contribute to Quantum Voyager

Quantum Voyager is an open-source project and we welcome contributions from developers, physicists, educators, and enthusiasts!

Ways to Contribute

  • Code: Implement new features, fix bugs, improve performance
  • Physics: Help improve the accuracy of our quantum simulations
  • Documentation: Write tutorials, improve explanations
  • Design: Enhance UI/UX, create visualizations
  • Testing: Report bugs, test on different platforms
  • Ideas: Suggest new experiments or features

Getting Started

  1. Fork the repository at github.com/Shahpure15/Quantum-Voyager
  2. Clone your fork locally and follow setup instructions in the README
  3. Check the Issues tab for beginner-friendly tasks or create a new Issue
  4. Make your changes in a new branch
  5. Submit a Pull Request with a clear description of your changes

Development Stack

  • Frontend: HTML, CSS, JavaScript
  • 3D Rendering: Three.js
  • CSS Framework: Tailwind CSS
  • Package Manager: npm
  • Development Server: live-server
